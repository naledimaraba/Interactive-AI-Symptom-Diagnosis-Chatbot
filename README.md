# Interactive AI Symptom Diagnosis Chatbot

This project is a rule-based AI chatbot that helps users identify possible health conditions based on symptoms.

The chatbot uses decision logic, severity scoring, and user interaction to simulate basic AI reasoning.

## Features
- Menu-based interaction
- Symptom detection
- Severity scoring (1–3)
- User feedback system
- Chat history logging

## Technologies Used
Python
Artificial Intelligence (Rule-Based Logic)
User Interaction Design

## Project Purpose
This project demonstrates how AI logic can be built using Python without machine learning.

## How to Run the Project
1. Download the file
2. Open in Python
3. Run the script
4. Choose symptoms from the menu
